# SPEC_H_01 — 产品规格书 (PRODUCT.md)

## 阶段 H: 团队协作工作台

**版本:** 1.0
**日期:** 2026-07-22
**状态:** 开发完成

---

## 1. 产品概述

阶段 H 为 ProjectPack Office Agent 引入完整的团队协作工作台功能，允许团队在平台内通过角色权限体系进行安全的项目协作，包括任务看板、风险管理、报告中心、评论讨论和通知收件箱。

### 核心价值
- **安全协作**: 项目级角色权限体系确保数据安全
- **统一工作台**: 一个入口查看进度、风险、报告、通知
- **可追溯**: 所有操作有记录、有审批流

---

## 2. 功能模块

### H.1 — 认证与角色系统
| 功能 | 描述 |
|------|------|
| 登录 | 用户名+密码 → JWT Bearer Token（24h 有效期） |
| 用户角色 | `admin` > `pm` > `member` > `guest` |
| 令牌验证 | HMAC-SHA256 自签名 Token，无外部依赖 |
| 账户管理 | 4 个演示用户预置 |

**演示账户:**
| 用户名 | 密码 | 显示名 | 角色 |
|--------|------|--------|------|
| admin | admin123 | 管理员 | admin |
| pm | pm123 | 项目经理 | pm |
| member | member123 | 团队成员 | member |
| guest | guest123 | 只读访客 | guest |

### H.2 — 项目成员管理
- 管理员可添加/移除成员、变更角色
- 成员列表按角色重要性排序
- 支持角色变更（admin 独有权限）

### H.3 — 项目总览看板
- 任务统计（按状态分组 + 总数）
- 风险统计（按严重度分组：活跃风险数）
- 待确认数量
- 最近文档变更（最近5条）
- 最近运行记录（最近5条）

### H.4 — 任务看板
- 按状态/负责人/优先级/日期范围筛选
- 支持多字段排序（due_date, priority, title, status）
- 支持分组视图（按负责人/状态/优先级）
- 关键字搜索

### H.5 — 风险中心
- 风险记录列表（可按严重度/生命周期筛选）
- 指派风险负责人（PM+ 权限）
- 风险生命周期操作：`acknowledge` → `resolve` / `dismiss` → `reopen`
- 每次操作自动通知负责人

### H.6 — 报告中心
- 报告草稿创建、编辑、版本管理
- 提交审批 → PM/Admin 审批/驳回/要求修改
- 审批历史可追溯
- 导出 PDF（reportlab）和 DOCX（python-docx）

### H.7 — 评论与@提及
- 多实体评论：`task` / `risk` / `report_section` / `report`
- 支持线程回复（parent_id）
- `@username` 提及成员，自动通知被提及者
- PM+ 可标记评论为已解决
- 作者可编辑自己的评论，PM+ 可删除评论

### H.8 — 通知收件箱
- 站内通知：提及、回复、风险指派、报告审批等
- 未读计数、标记已读、全部已读
- 分页查询
- **不直接外发 IM**（仅站内通知）

### H.9 — 文件下载权限
- 文件下载需认证 + member+ 角色
- 路径遍历保护
- 上传接口保持原有行为
